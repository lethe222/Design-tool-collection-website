
# Design tool collection website

Design tools collection site built with React and Vite.

Original design source:
[Figma file](https://www.figma.com/design/6OKeqLpjPeTfaAuCNnKuCq/Design-tool-collection-website)

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Deployment

This project is configured for GitHub Pages at:

`https://lethe222.github.io/Design-tool-collection-website/`

Pushes to the `main` branch trigger the GitHub Actions workflow in
[.github/workflows/deploy.yml](/Users/chao/Desktop/github/Design tool collection website/.github/workflows/deploy.yml).
